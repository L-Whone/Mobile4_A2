**A2: AR Exploration**

&#x09;This Project is a mockup of a Pokémon Go Demo, You can Spawn Monsters, and throw capture balls at them.



**How to Open:**

&#x09;There are two ways to test the game out,

1. &#x20;**Android/PC**
* Download the Project
* Open SampleScene in Scenes Folder
* Press Play
* Look around with RMB to allow XR environment to scan surroundings
* Switch Game Mode to Simulator (top left of Game View)
* Spawn a Monsters
* Spawn a Capture ball
* Throw with ball with mouse in Simulator Mode



**2. iPhone**

* Do Same as above
* Install Unity Remote 5 on iPhone
* Open Project Settings -> Editor -> Device -> Any iOS Device
* Connect iPhone to Computer with iTunes App installed
* Switch Build Profiles to iOS
* Press play, allow XR environment to scan surroundings
* Spawn Monsters and Capture on your iPhone
